# Исследование популярных жанров

В этой директории собраны результаты небольших исследований по популярности жанров в разных медиа:

- Книги — см. [books.md](./books.md) (ориентир для структуры и методологии)
- Книги (исходный список из задачи) — см. корневой файл [`genres/list.md`](../../genres/list.md)
- Кино и сериалы — см. [movies_series.md](./movies_series.md)
- Игры — см. [games.md](./games.md)
- Сводное сравнение по медиа — см. [summary.md](./summary.md)

Материалы ориентируются на публичные агрегаты статистики (The Numbers, Statista, IMDb/TV, Steam, отчёты Newzoo и др.). Для каждого раздела приведены краткое описание методики, список жанров и ссылки на источники.

## Как обновлять/расширять
- При добавлении источников фиксируйте год/регион и метрику (доля кассы, продажи, MAU и т.п.).
- Схожие поджанры объединяйте в кластеры для сопоставимости между медиа.
- Чётко помечайте места с качественными выводами без точных процентов, чтобы избежать ложной точности.
