# Популярные жанры: книги

Этот файл добавлен для полноты картины в основной ветке. В исходной задаче упоминался уже существующий файл в ветке `researches`. Здесь — краткая сводка популярных книжных жанров и ориентиры по источникам.

## Ключевые жанры (обобщённо)

- Fantasy / Фэнтези
- Romance / Романы
- Mystery & Thriller / Детектив и триллер
- Science Fiction / Научная фантастика
- Young Adult / Подростковая литература (как макро‑категория)
- Historical Fiction / Историческая проза
- Non‑Fiction / Нон‑фикшн (биографии, саморазвитие и др.)
- Horror / Ужасы

Популярность жанров отличается по рынкам (США/ЕС/Россия и т.д.) и по метрике (продажи, чарты, библиотеки/читалки, премии, рейтинги).
